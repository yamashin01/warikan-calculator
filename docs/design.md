# ワリカン計算機 - 設計書

## 1. 機能要件

### 1.1 基本機能
- **合計金額入力**: ユーザーが支払う合計金額を円単位で入力
- **人数入力**: 割り勘する人数を整数で入力
- **計算実行**: 「計算する」ボタンで一人当たりの金額を算出
- **結果表示**: 計算結果を分かりやすく表示

### 1.2 追加要件
- 入力値のバリデーション
- エラーメッセージの表示
- 即座にクリアできる機能

## 3. 入出力仕様

### 3.1 入力
| 項目 | 型 | 範囲 | 必須 |
|------|-----|------|------|
| 合計金額 | 整数 | 1以上 | ○ |
| 人数 | 整数 | 1以上 | ○ |

### 3.2 出力
| 項目 | 型 | 備考 |
|------|-----|------|
| 一人当たりの金額 | 整数 | 100円単位で切り上げ |

---

## 4. バリデーションルール

### 4.1 エラーケース
1. **空欄**: 「合計金額と人数を入力してください」
2. **非数値入力**: 「数値を入力してください」
3. **0以下の値**: 「1以上の値を入力してください」
4. **小数点（人数）**: 「人数は整数を入力してください」

### 4.2 エラー表示
- 赤色のアラートボックスで画面上部に表示
- エラーをクリアするか、入力を修正すると自動で消える

## 5. 計算ロジック

```
一人当たりの金額 = 合計金額 ÷ 人数 → 100円単位で切り上げ

例:
- 合計金額: 3000円
- 人数: 4人
- 計算: 3000 ÷ 4 = 750円 → 750円（既に100円単位）
- 結果: 750円

- 合計金額: 10000円
- 人数: 3人
- 計算: 10000 ÷ 3 = 3333.33...円 → 3400円（100円単位で切り上げ）
- 結果: 3400円

- 合計金額: 1000円
- 人数: 3人
- 計算: 1000 ÷ 3 = 333.33...円 → 400円（100円単位で切り上げ）
- 結果: 400円
```

## 6. 技術仕様

- **HTML5**: 構造

- **バニラJS**: ロジック（インライン）

```
  - Tailwind CSS CDN読み込み
  - JavaScript（<script>タグ内）
```

- 入力値の取得 (`document.getElementById`)

- 計算実行

## 7. ユーザーフロー

```
1. ページを開く
2. 合計金額を入力
3. 人数を入力
4. 「計算する」ボタンをクリック
   ├─ バリデーション OK → 結果表示
   └─ バリデーション NG → エラー表示
5. 「クリア」ボタンで入力と結果をリセット
```
