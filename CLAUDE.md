## プロジェクト概要

**ワリカン計算機** - シンプルなWebアプリケーション。割り勘費用の計算に特化したツール。

- **合計金額** と **人数** を入力して、**一人当たりの金額** を計算表示
- 基本機能のみ（不均等割り勘や詳細な支出管理は非対象）
- 完全にシングルファイル実装（外部ビルドプロセスなし）

## 開発のポイント

### 1. Tailwind CSS の使用
- CDN経由で読み込み（外部依存最小化）
- Tailwind CDN: `<script src="https://cdn.tailwindcss.com"></script>`
- カスタム設定なし（デフォルト設定のみ）

### 2. JavaScript ロジック
- すべてのロジックは `<script>` タグ内（インライン）
- 外部ライブラリ不使用（バニラJS）
- DOM操作は `getElementById` と `addEventListener` を使用
- 主要な処理：入力バリデーション → 計算 → DOM更新

### 3. バリデーション実装
`calculate()` 関数内で以下の順序でチェック：
1. 空欄チェック（`value.trim() === ''`）
2. 数値変換可能チェック（`isNaN()`）
3. 値の範囲チェック（`<= 0`）
4. 人数の整数チェック（`Number.isInteger()`）

### 4. 100円単位での切り上げ処理
```javascript
const perPerson = totalNum / headNum;
const rounded = Math.ceil(perPerson / 100) * 100;  // 100円単位で切り上げ
```
- `perPerson / 100` で100円単位に正規化
- `Math.ceil()` で切り上げ
- `* 100` で元の単位に戻す

## テスト方法

**自動テストなし** - ブラウザで手動確認が標準。

### テストシナリオ
1. **基本計算**: 3000円 ÷ 4人 = 750円（既に100円単位）
2. **切り上げ対応**: 10000円 ÷ 3人 = 3400円（333.33...→切り上げ）
3. **端数切り上げ**: 1000円 ÷ 3人 = 400円（333.33...→切り上げ）
4. **空欄エラー**: 何も入力しないと「合計金額と人数を入力してください」
5. **値の範囲エラー**: -100円など0以下を入力するとエラー表示
6. **クリア機能**: 入力と結果がリセットされる

**テスト実行方法**:

## よくある修正パターン

### エラーメッセージを追加する場合
1. `showError()` 関数で表示処理が一元化されているので、メッセージテキストのみ変更
2. `docs/design.md` の「バリデーションルール」も併せて更新

### UIのスタイルを変更する場合
1. Tailwind CSS クラスのみで対応（インラインCSS追加は避ける）
2. 色の変更例: `bg-blue-600` → `bg-green-600`
3. 間隔の変更例: `p-8` → `p-10`

### 計算ロジックを変更する場合
1. `calculate()` 関数内の計算式を修正
2. バリデーション処理には影響させない
3. 結果表示ロジック（`resultAmount.textContent`）も必要に応じて調整
